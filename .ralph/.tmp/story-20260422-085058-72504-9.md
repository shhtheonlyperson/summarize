### LLR-009: Show Local-Only Status in Side Panel
Status: in_progress
Depends on: LLR-008

Description:
Make the side panel visibly communicate whether a run will stay local or use a remote provider, and surface local runtime readiness problems.

Acceptance Criteria:
- [ ] Add a compact local runtime/privacy status surface to the side panel.
- [ ] Show local-only enabled, local runtime reachable, selected route/model, and actionable setup errors.
- [ ] Avoid exposing secrets or raw environment values in the UI.
- [ ] Add extension tests for local status rendering and privacy warning behavior.
- [ ] Run pnpm -C apps/chrome-extension build and supported extension tests when feasible.
