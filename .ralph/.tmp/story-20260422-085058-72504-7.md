### LLR-007: Add Local-Only Privacy Guard
Status: in_progress
Depends on: LLR-006

Description:
Add a hard privacy mode that prevents summarization requests from using cloud providers unless the user explicitly disables local-only mode.

Acceptance Criteria:
- [ ] Add config/request support for local-only mode using existing config patterns.
- [ ] When local-only mode is enabled, reject or reroute non-local model/provider selections before any remote provider request is made.
- [ ] Return clear errors that explain which setting caused the remote provider to be blocked.
- [ ] Add tests proving known remote providers are blocked in local-only mode.
- [ ] Add tests proving localhost/OpenAI-compatible local endpoints remain allowed.
- [ ] Run targeted tests plus pnpm -s check.
