### LLR-011: Add Local Research Memory Design
Status: in_progress
Depends on: LLR-001

Description:
Design durable local research memory without implementing storage yet. The design should choose a storage backend and describe how runs, sources, artifacts, prompts, model routes, and failures will be stored locally.

Acceptance Criteria:
- [ ] Add a design document for durable local research memory.
- [ ] Compare Postgres, SQLite, and existing cache/session storage for this fork's local-first goals.
- [ ] Define initial entities for runs, sources, artifacts, events, model route metadata, and privacy mode metadata.
- [ ] State which data must never leave localhost in local-only mode.
- [ ] Do not implement the database in this story.
