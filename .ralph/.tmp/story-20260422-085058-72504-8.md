### LLR-008: Expose Local Runtime Status in Daemon API
Status: in_progress
Depends on: LLR-004, LLR-007

Description:
Expose daemon endpoints for local runtime status so the side panel can explain readiness, model routing, and local-only privacy state.

Acceptance Criteria:
- [ ] Add a token-authenticated daemon endpoint for local runtime status/probes.
- [ ] Endpoint response must include local-only mode, configured runtime type, endpoint host, selected model hints, and probe status.
- [ ] Endpoint must not leak API keys or sensitive environment values.
- [ ] Add daemon tests with mocked probe logic.
- [ ] Run daemon tests plus pnpm -s typecheck.
