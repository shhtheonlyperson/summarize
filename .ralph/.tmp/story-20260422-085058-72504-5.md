### LLR-005: Add CLI Local Runtime Probe Command
Status: in_progress
Depends on: LLR-004

Description:
Expose local runtime diagnostics in the CLI so users can verify their local LLM setup before launching the extension or daemon.

Acceptance Criteria:
- [ ] Add a CLI command for probing configured local runtimes.
- [ ] The command must print clear success, warning, and failure messages for local endpoints.
- [ ] The command must support JSON output for tests and scripts if the CLI has existing JSON output conventions.
- [ ] Add CLI tests that mock probe results and do not require a live model server.
- [ ] Update docs with the probe command.
- [ ] Run targeted CLI tests plus pnpm -s typecheck.
