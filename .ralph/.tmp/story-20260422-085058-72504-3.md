### LLR-003: Add Local Runtime Registry Types
Status: in_progress
Depends on: LLR-002

Description:
Introduce typed local runtime registry primitives for llama.cpp-compatible OpenAI endpoints and Ollama without changing default behavior. The registry should be usable by CLI, daemon, and extension code.

Acceptance Criteria:
- [ ] Add typed runtime descriptors for OpenAI-compatible, llama.cpp, and Ollama local runtimes.
- [ ] Add defaults that prefer 127.0.0.1 localhost endpoints and avoid remote URLs unless explicitly configured.
- [ ] Add validation helpers for local runtime base URLs.
- [ ] Add unit tests for runtime descriptor parsing and URL validation.
- [ ] Run targeted tests plus pnpm -s typecheck.
